# cpp_arcade
## An arcade game simulator

### Games
We currently support a few games.
- Snake
- Nibbler
- Blockade
- Centipede
- SolarFox
- Pacman
- Pong (multiplayer, network)
- Space Invaders

### Libs
We currently support those libraries
- Allegro5
- SDL2
- SFML
- XLib
- Ncurses
- Libcaca
- LibLapin

You can switch to another library at any moment.
